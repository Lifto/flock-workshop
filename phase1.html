1 — Install build tools
sudo dnf install -y cmake make gcc git gcc-c++ libcurl-devel
2 — Get files from USB
mkdir ~/chatbot && cd ~/chatbot
cp /run/media/$USER/DISK_IMG/* .
tar xzf llama.cpp-b7783.tar.gz
At home: git clone --branch b7783 --depth 1 https://github.com/ggerganov/llama.cpp
3 — Build
cd ~/chatbot/llama.cpp
mkdir build && cd build
cmake .. -DLLAMA_BUILD_EXAMPLES=ON
cmake --build . --config Release
⏲ 5–10 min build. Help a neighbor while you wait.
4 — Chat!
cd ~/chatbot/llama.cpp/build
./bin/llama-cli \
  -m ~/chatbot/microsoft_Phi-4-mini-instruct-Q4_K_M.gguf \
  -c 4096 \
  -p "You are a helpful assistant."
You see a > prompt. Ask it something.
Type /exit to quit.
5 — Change the prompt
./bin/llama-cli \
  -m ~/chatbot/microsoft_Phi-4-mini-instruct-Q4_K_M.gguf \
  -c 4096 \
  -p "You are a mass-market paperback
romance novelist. Answer everything
as if narrating a scene."
Same model, different -p, different personality.
Try: pirate, Shakespearean scholar, a cat who learned to type…
